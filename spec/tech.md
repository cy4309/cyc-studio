# Tech Stack

## Framework

- Next.js 16 (App Router)
- React 18
- TypeScript

## Styling

- Tailwind CSS only
- No inline styles
- No CSS-in-JS libraries

## Animation

- Framer Motion for UI motion
- Lenis for smooth scrolling
- GSAP is optional and should not be introduced unless specified

## Rendering

- Prefer Server Components for layout
- Use Client Components only when interaction is required

## Deployment

- Vercel

## Restrictions

- Do not introduce Redux
- Do not introduce class components
- Do not introduce unnecessary libraries
