# CYC Studio

CYC Studio is a personal creative frontend studio website.

## Purpose

- Showcase selected interactive and visual web projects
- Demonstrate creative frontend engineering capability
- Act as a professional portfolio for client work

## Owner

- Frontend engineer focused on interaction, motion, and Web-based experiences

## Target Audience

- Design-aware clients
- Creative directors
- Tech-driven brands
- People familiar with Apple-level polish and CSS Awards websites

## Tone

- Confident
- Calm
- Precise
- Minimal

## Non-Goals

- No blog system
- No CMS in phase 1
- No marketing-heavy copy
- No backend-heavy features
